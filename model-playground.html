How It Works

This model uses a DINOv2 vision transformer pretrained with self-supervised learning. Here's what happens:

  1. Feature Extraction: The DINOv2 backbone extracts rich visual features from the image
  2. Classification: A learned classification head predicts disease categories
  3. Attention: Self-attention mechanisms identify relevant image regions

Architecture: DINOv2-ViT-S/14 + Custom Classification Head
Training: Fine-tuned on 30K+ plant disease images
Accuracy: 94% on test set
