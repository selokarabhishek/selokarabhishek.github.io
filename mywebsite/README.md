# Abhishek Selokar - Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer and AI/ML Engineer.

## Features

- **Modern Design**: Clean, professional design with smooth animations and transitions
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Dynamic typing animation
  - Smooth scroll navigation
  - Animated statistics counters
  - Interactive particle effects
  - Scroll-to-top button
- **Sections**:
  - Hero section with animated introduction
  - About me with statistics
  - Technical skills showcase
  - Featured projects gallery
  - Contact form with social links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Animations)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Poppins, Fira Code)

## Color Scheme

- Primary: `#6366f1` (Indigo)
- Secondary: `#8b5cf6` (Purple)
- Accent: `#ec4899` (Pink)
- Dark Background: `#0f172a`
- Darker Background: `#020617`

## Setup

1. Clone the repository:
```bash
git clone https://github.com/selokarabhishek/selokarabhishek.github.io.git
```

2. Open `index.html` in your browser or deploy to GitHub Pages

## Customization

To customize this portfolio for your own use:

1. Update personal information in `index.html`
2. Modify color scheme in CSS variables in `styles.css`
3. Add your own projects in the Projects section
4. Update social media links
5. Replace placeholder images with your own photos

## GitHub Pages Deployment

This site is designed to work with GitHub Pages. To deploy:

1. Push your changes to the `main` branch
2. Go to repository Settings > Pages
3. Select `main` branch as source
4. Your site will be live at `https://selokarabhishek.github.io`
